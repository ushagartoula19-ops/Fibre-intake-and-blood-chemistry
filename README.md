# NHANES Analysis: Fiber, Glucose, and Lipid Profile

## 📌 Project Overview
This project analyzes the relationship between dietary fiber intake and metabolic health markers using NHANES (National Health and Nutrition Examination Survey) data.

## 📊 Research Question
Is higher dietary fiber intake associated with:
- Lower blood glucose levels?
- Improved lipid profile (HDL, Total Cholesterol, Triglycerides)?

## 🗂️ Data Sources
NHANES datasets used:
- DEMO_L.xpt (Demographics)
- DR1TOT_L.xpt (Dietary fiber intake)
- GLU_L.xpt (Blood glucose)
- HDL_L.xpt (HDL cholesterol)
- TECHOL_L.xpt (Total cholesterol)
- TRIGLY_L.xpt (Triglycerides)

👉 Data can be downloaded from: https://wwwn.cdc.gov/nchs/nhanes/

## ⚙️ Methods
- Loaded SAS XPT files using Python
- Selected relevant variables
- Merged datasets using participant ID (SEQN)
- Handled missing values
- Prepared dataset for analysis

## 🛠️ Tools Used
- Python
- pandas
- pyreadstat
- Jupyter Notebook
